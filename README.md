# Plantilla novela — Sistema disciplinario

Este repositorio funciona como eje estructural de cada novela.

Regla fundamental:
Nada está consolidado hasta que está en este repositorio.

Estructura:
- 00_metodo → reglas del sistema
- 01_manuscrito → texto consolidado
- 02_ledger → estado técnico
- 03_personajes → fichas
- 04_cronologia → línea temporal
- 05_decisiones → decisiones cerradas
- 06_export → archivos preparados para IA o edición externa
